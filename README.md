# LLM Inference Cost Simulation

This Python script simulates the inference cost of a Large Language Model (LLM) based on hypothetical per-token costs, average prompt/completion lengths, and request traffic. It demonstrates how factors like total token count and requests per second contribute significantly to the overall operational expense of running an LLM, illustrating the importance of token economics.

## Language

`python`

## How to Run

1. Save the code as `llm_inference_cost.py`.
2. Run from your terminal: `python llm_inference_cost.py`
3. Modify the `_PARAMETERS` at the top of the script to explore different cost scenarios.
